1.) Install requirements.txt
2.) add your [Are.na API](https://dev.are.na/documentation) ARENA_ACCESS_TOKEN and your ARENA_USER_ID to an environmental variable file
3.) run `python arena_vector_store.py` to create an embedding map of your are.na channels
4.) run `parena_search.py` to serach text via CLI and `arena_search_gui.py` to drag and drop pdfs
